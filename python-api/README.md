##Running Tests
```bash
cd python-api
```

####Run Unittests
```bash
make unittests
```

####Run Integration Tests
```bash
make integration
```

####Lint and pep8 Tests
```bash
make lint
make pep8
```
####Running everything together
```bash
make test
```
